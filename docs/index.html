wav.js

Provides some wrapper details for the concept of .wav data to allow metadata to be easily accessed. Credit to http://soundfile.sapp.org/doc/WaveFormat/ for the information about .wav file formatting needed. This file assumes little endian data and that the data chunk is the last chunk in the file.
