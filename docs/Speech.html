Class: Speech

Speech(audio)

Aurora's speech object. This high level object provides a wrapper for an Aurora AudioFile object that contains speech audio as well as listening functions for creating new Speech objects from available audio equipment and conversion methods to obtain a promise for the Aurora Text version of the audio.
