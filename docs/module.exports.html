exports(inputBuffer, fixHeaderErrorsopt)

A class holding a buffer needed to extract meaningful data from .wav data

Constructor

new exports(inputBuffer, fixHeaderErrorsopt)

Copies and validates the input buffer and creates a WavBuffer from it.
Parameters:
Name Type Attributes Default Description
inputBuffer Buffer The buffer to use.
fixHeaderErrors boolean <optional>
 false Frequently, .wav headers will be malformed. If this is true, the constructor will attempt to fix conflicting header information. If false, errors will be thrown for malformed headers.
